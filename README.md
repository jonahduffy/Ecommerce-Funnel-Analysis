# E-Commerce Sales Funnel Analysis

## Project Overview
This project analyzes the customer journey of an e-commerce platform using SQL and Tableau. By examining user event data, I identified key drop-off points in the sales funnel, analyzed the most profitable traffic sources, and measured the average time to conversion.

## The Data
The dataset used is a simulated e-commerce event log (`user_events`) containing user interactions from initial page views to final purchases. 

## Key Business Questions Answered
1. What is the overall conversion rate from website visitor to paying customer?
2. At which stage of the checkout process are we losing the most users?
3. Which marketing channels drive the highest quality traffic (highest purchase conversion)?
4. How long does it take an average user to complete a purchase?

## Insights & Recommendations
* **Friction at the Top:** Only 31% of users add items to their cart, indicating a need for better product discovery or stronger calls-to-action on product pages.
* **Email is King:** While Social Media drives the most raw traffic, Email marketing has a massive 33.9% purchase conversion rate. Marketing spend should be reallocated to grow the email subscriber list.
* **Impulse Buying:** The average time from first view to purchase is only 24.6 minutes, highlighting a highly impulsive customer base.
